# React frontend (Vite)

This folder contains the React version of the Field Notes Todo UI.

## Run

1. Start backend:
   - Use VS Code Run on `TodoApplication.java` **or** start your Spring Boot app on port 8080.
2. Start React:
   - `cd frontend/react-app`
   - `npm install`
   - `npm run dev`

Open the URL shown by Vite (typically http://localhost:5173).

The app calls the backend at:
- `http://localhost:8080/api/tasks`
